# 一、归一化  
## 简单来说：  

&emsp;&emsp; 归一化就是要把需要处理的数据经过处理后（通过某种算法）限制在你需要的一定范围内，使得没有可比性的数据变得具有可比性，但又还会保持相比较的两个数据之间的相对关系。  

<br/>
## 意义：  

&emsp;&emsp;归一化的依据非常简单，不同变量往往量纲不同，通过归一化可以消除量纲对最终结果的影响，使不同变量具有可比性，可使后面数据处理的方便，也保证程序运行时收敛加快。  

&emsp;&emsp;归一化是一种简化计算的方式，即将有量纲的表达式，经过变换，化为无量纲的表达式，成为纯量。  

## 方式  

#### 1. min-max标准化/离差标准化  

&emsp;&emsp;通过对原始数据进行线性变换把数据映射到[0,1]（便于统计分析）之间。  
变换函数为：
$$x'=\frac { x-min }{ max-min } $$  

其中minmin是样本中最小值，maxmax是样本中最大值。注意在数据流场景下最大值与最小值是变化的,需要实时的去更新。  

#### 2. Z-score标准化方法  

&emsp;&emsp; 这种方法给予原始数据的均值（mean）和标准差（standard deviation）进行数据的标准化。经过处理的数据符合标准正态分布，即均值为0，标准差为1。  

转化函数为：
$$x'=\frac { x-\mu  }{ \sigma  } $$  

其中$\mu$为所有降本数据的均值，$\sigma$为所有样本数据的标准差。
